Overview
The Appointment Management App is a standalone Java program that allows users to manage various types of appointments. It supports one-time, daily, and monthly appointments, offering features to create, update, delete, and retrieve appointments. The program is implemented with a GUI for easy user interaction and utilizes unit tests for reliable functionality.

Features
Appointment Types:

One-time Appointment: Occurs on a specific date.
Daily Appointment: Recurs daily over a date range.
Monthly Appointment: Recurs monthly within a date range.
Core Functionalities:

Add new appointments.
Delete existing appointments.
Update existing appointments with new details.
Retrieve appointments by date and sort them.
User Interface:

Easy-to-use GUI built with Swing.
Supports creating, displaying, updating, and deleting appointments.
Error handling with user-friendly dialogs.
